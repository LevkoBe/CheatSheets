üî¥ Theorems Without Proofs

Handshaking Lemma
For any graph \(G = (V, E)\): \(\sum_{v \in V} \deg(v) = 2|E|\)
Corollary 1: Number of vertices with odd degree is even
Corollary 2: Average degree = \(\frac{2|E|}{|V|}\)
Bipartite Graph Characterization
Graph \(G\) is bipartite ‚ü∫ \(G\) contains no odd cycles
Powers of Adjacency Matrix
For adjacency matrix \(A\): \((A^k)_{ij}\) = number of walks of length \(k\) from vertex \(i\) to vertex \(j\)
Chromatic Number Bounds
Lower bound: \(\chi(G) \geq \omega(G)\) (clique number)
Upper bound: \(\chi(G) \leq \Delta(G) + 1\) (max degree + 1)
Improved: \(\chi(G) \geq \frac{n}{\alpha(G)}\) (independence number)
Brooks' Theorem
If \(G\) is connected and neither complete nor an odd cycle, then \(\chi(G) \leq \Delta(G)\)
Eulerian Graph Characterization
For connected graph \(G\), the following are equivalent:
‚Ä¢ \(G\) has Eulerian cycle
‚Ä¢ All vertices have even degree
‚Ä¢ Edge set can be partitioned into cycles
Havel-Hakimi Algorithm
Input: Degree sequence d‚ÇÅ ‚â• d‚ÇÇ ‚â• ... ‚â• d‚Çô 1. If all degrees are 0, return "graphic" 2. If d‚ÇÅ ‚â• n or d‚ÇÅ < 0, return "not graphic" 3. Remove d‚ÇÅ, subtract 1 from next d‚ÇÅ terms 4. Reorder and repeat

üî¥ More Theorems Without Proofs

Erd≈ës-Gallai Theorem
Sequence \(d_1 \geq d_2 \geq \ldots \geq d_n\) is graphic ‚ü∫
1. \(\sum_{i=1}^n d_i\) is even, and
2. For all \(1 \leq k \leq n\): \[\sum_{i=1}^k d_i \leq k(k-1) + \sum_{i=k+1}^n \min(d_i, k)\]
Kuratowski's Theorem
Graph \(G\) is planar ‚ü∫ \(G\) contains no subdivision of \(K_5\) or \(K_{3,3}\)
Whitney Isomorphism Theorem
For graphs \(G_1, G_2\) (neither \(K_3\) nor \(K_{1,3}\)):
\(G_1 \cong G_2\) ‚ü∫ \(L(G_1) \cong L(G_2)\)
where \(L(G)\) is the line graph of \(G\)
Block Graph Characterization
Graph \(G\) is a block graph ‚ü∫ \(G\) is a chordal graph with no induced \(K_4\)
Equivalently: Every block is a clique
Handshaking Lemma for Digraphs
For any digraph \(D = (V, A)\): \[\sum_{v \in V} d^+(v) = \sum_{v \in V} d^-(v) = |A|\]
Digraph Hamiltonian Generalization
Meyniel's Theorem: If for all non-adjacent vertices \(u, v\): \(d^+(u) + d^+(v) + d^-(u) + d^-(v) \geq 2n - 1\)
then the digraph is Hamiltonian.

üü¢ Theorems With Proofs

Radius-Diameter Construction
Theorem: For any integers \(r, d\) with \(r \leq d \leq 2r\), there exists a graph \(G\) with \(\text{rad}(G) = r\) and \(\text{diam}(G) = d\).

Proof (Construction):
Start with path \(P\) of length \(d\): \(v_0 - v_1 - \ldots - v_d\)
Add vertex \(c\) connected to \(v_r\) (this will be in center)
Verify: \(e(c) = \max\{r, d-r\} = r\) (since \(d \leq 2r\))
For any \(v_i\): \(e(v_i) = \max\{i+1, d-i+1\} \geq r+1\)
Thus \(\text{rad}(G) = r\) and \(\text{diam}(G) = d\) ‚àé
Center Realization
Theorem: Any graph \(H\) can be realized as the center of some connected graph \(G\).

Proof (Construction):
Let \(H\) have \(n\) vertices
For each vertex \(v \in V(H)\), attach path of length \(n+1\)
Call resulting graph \(G\)
For \(v \in V(H)\): \(e_G(v) = n+1\)
For pendant vertices: eccentricity \(> n+1\)
For internal path vertices: eccentricity \(> n+1\)
Therefore \(\text{center}(G) = V(H)\) ‚àé
Five Color Theorem
Theorem: Every planar graph is 5-colorable.

Proof:
By induction on \(|V|\). Base case: \(|V| \leq 5\) is trivial
Assume theorem holds for all planar graphs with \(< n\) vertices
By Euler's formula: \(\sum \deg(v) = 2|E| \leq 6|V| - 12\)
Average degree \(< 6\), so \(\exists v\) with \(\deg(v) \leq 5\)
Remove \(v\), color \(G - v\) with 5 colors by IH
If \(\deg(v) \leq 4\), color \(v\) with unused color
If \(\deg(v) = 5\), use Kempe chain argument to free a color ‚àé

üü¢ More Proofs

Ore's Theorem
Theorem: If \(G\) is simple with \(n \geq 3\) vertices and \(\deg(u) + \deg(v) \geq n\) for all non-adjacent \(u, v\), then \(G\) is Hamiltonian.

Proof:
Let \(P = v_1v_2\ldots v_k\) be longest path in \(G\)
All neighbors of \(v_1\) and \(v_k\) are on \(P\)
Let \(N(v_1) = \{v_{i_1}, \ldots, v_{i_s}\}\) where \(i_1 < \ldots < i_s\)
If \(v_{i_j} \in N(v_1)\) then \(v_{i_j-1} \notin N(v_k)\) (else cycle)
Since \(|N(v_1)| + |N(v_k)| \geq n\) and \(|P| \leq n\), get contradiction
Therefore \(k = n\) and \(P\) is Hamiltonian path
Connect endpoints using degree condition to get cycle ‚àé
Dirac's Theorem: If \(\deg(v) \geq n/2\) for all \(v\), then \(G\) is Hamiltonian.
Proof: For non-adjacent \(u, v\): \(\deg(u) + \deg(v) \geq n\). Apply Ore's theorem.
Connected Graph Realization
Theorem: Graphic sequence \(M\) is realizable by connected graph ‚ü∫ \(M\) is graphic and \(\sum_{d \in M} d \geq 2(|M| - 1)\).

Proof:
(\(\Rightarrow\)) Connected graph on \(n\) vertices has \(\geq n-1\) edges
By handshaking: \(\sum d = 2|E| \geq 2(n-1)\)
(\(\Leftarrow\)) Use Havel-Hakimi to construct graph \(G\)
If \(G\) disconnected with components \(C_1, \ldots, C_k\)
Then \(|E| = \sum_{i=1}^k |E(C_i)| \leq \sum_{i=1}^k (|C_i| - 1) = n - k\)
So \(\sum d = 2|E| \leq 2(n-k) < 2(n-1)\), contradiction ‚àé
Euler's Polyhedron Formula
Theorem: For connected planar graph: \(v - e + f = 2\).

Proof:
Induction on \(e\). Base: \(e = 0 \Rightarrow v = 1, f = 1\). Check: \(1 - 0 + 1 = 2\) ‚úì
Inductive step: Remove edge \(e\)
Case 1: \(e\) is bridge. Then \(v' = v\), \(e' = e-1\), \(f' = f-1\)
By IH: \(v' - e' + f' = 2 \Rightarrow v - (e-1) + (f-1) = 2\) ‚úì
Case 2: \(e\) not bridge. Then \(v' = v\), \(e' = e-1\), \(f' = f-1\)
By IH: \(v - (e-1) + (f-1) = 2 \Rightarrow v - e + f = 2\) ‚úì ‚àé
Intersection Graph Universality
Theorem: Every graph is the intersection graph of some family of sets.

Proof (Construction):
Given graph \(G = (V, E)\)
For each vertex \(v \in V\), define \(S_v = \{e \in E : v \in e\}\)
Consider intersection graph \(H\) of family \(\{S_v : v \in V\}\)
Vertices \(u, v\) adjacent in \(H\) ‚ü∫ \(S_u \cap S_v \neq \emptyset\)
This happens ‚ü∫ \(\exists e \in E\) with \(u, v \in e\)
This happens ‚ü∫ \(u, v\) adjacent in \(G\)
Therefore \(G \cong H\) ‚àé
Acyclic Digraph Source-Sink
Theorem: Every acyclic digraph has a source and a sink.

Proof:
Suppose no source exists (all vertices have indegree \(> 0\))
Start at any vertex \(v_0\)
Since \(d^-(v_0) > 0\), \(\exists v_1\) with \(v_1 \to v_0\)
Since \(d^-(v_1) > 0\), \(\exists v_2\) with \(v_2 \to v_1\)
Continue to build sequence \(v_0 \leftarrow v_1 \leftarrow v_2 \leftarrow \ldots\)
Since \(|V|\) finite, some vertex repeats: \(v_i = v_j\) for \(i < j\)
This creates directed cycle \(v_i \leftarrow v_{i+1} \leftarrow \ldots \leftarrow v_j = v_i\)
Contradiction! Similarly for sink ‚àé
R√©dei's Theorem
Theorem: Every tournament has a Hamiltonian path.

Proof:
Induction on \(n = |V|\). Base: \(n = 1, 2\) trivial
Assume true for tournaments on \(n-1\) vertices
Remove vertex \(v\), get tournament \(T'\) on \(n-1\) vertices
By IH, \(T'\) has Hamiltonian path \(P: u_1 \to u_2 \to \ldots \to u_{n-1}\)
Case 1: \(v \to u_1\). Then \(v \to u_1 \to \ldots \to u_{n-1}\) is Hamiltonian
Case 2: \(u_{n-1} \to v\). Then \(u_1 \to \ldots \to u_{n-1} \to v\) is Hamiltonian
Case 3: \(u_1 \to v\) and \(v \to u_{n-1}\). Find \(i\) with \(u_i \to v \to u_{i+1}\)
Then \(u_1 \to \ldots \to u_i \to v \to u_{i+1} \to \ldots \to u_{n-1}\) is Hamiltonian ‚àé
